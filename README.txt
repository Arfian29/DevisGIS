Proyek ini adalah aplikasi web yang dibangun dengan Tailwind CSS sebagai framework utama untuk styling yang efisien dan responsif. Proyek ini juga menggunakan Preline, sebuah pustaka komponen yang kompatibel dengan Tailwind, untuk menambah fungsionalitas UI yang interaktif. Untuk ikon, proyek ini memanfaatkan Feather Icons dan ikon dari SVG Repo, yang keduanya menyediakan aset grafis yang minimalis, modern, dan open-source. Dengan penggunaan teknologi-teknologi ini, aplikasi memiliki tampilan yang bersih, intuitif, dan siap untuk dikembangkan lebih lanjut.
